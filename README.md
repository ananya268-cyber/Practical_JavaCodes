# Practical_JavaCodes
This repository contains practical codes related to **Object-Oriented Programming (OOPs) concepts in Java**.  
It is designed to help students understand the basics of OOPs through simple examples.

---

Programs included:
- Classes and Objects
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
- Interfaces
- Exception Handling
- Stack and Arrays
- Matrix operations (Addition, Transpose)

---

How to Run
- Open the project in **IntelliJ IDEA** (or any Java IDE).  
- Go to the `src` folder.  
- Compile and run the desired `.java` file.
